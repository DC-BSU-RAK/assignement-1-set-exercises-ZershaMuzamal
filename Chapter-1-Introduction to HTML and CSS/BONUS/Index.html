HTML & CSS Quick Reference Guide

A concise guide to essential HTML tags and CSS properties.

Essential HTML Tags

<h1> to <h6> - Headings

Defines headings with different levels of importance.

<h1>Main Heading</h1>

<p> - Paragraph

Defines a block of text as a paragraph.

<p>This is a paragraph.</p>

<a> - Anchor

Creates a hyperlink to another page or resource.

<a href="https://example.com">Visit</a>

<img> - Image

Embeds an image into the page.

<img src="image.jpg" alt="Image">

<ul> & <ol> - Lists

Creates unordered and ordered lists.

<ul><li>Item</li></ul>

<table> - Table

Displays tabular data in rows and columns.

<table><tr><td>Data</td></tr></table>

<form> - Form

Creates a form to collect user input.

<form><input type="text"></form>

<button> - Button

Defines a clickable button element.

<button>Click me</button>

<div> - Division

Defines a division or section in the document.

<div>Content</div>

<span> - Inline Container

Defines a container for inline elements or text styling.

<span>Text</span>

Essential CSS Properties

color

Sets the text color of an element.

p { color: grey; }

background-color

Defines the background color of an element.

body { background-color: black; }

font-family

Specifies the font style for text content.

p { font-family: Arial; }

border

Applies a border around an element.

div { border: 1px solid white; }

padding

Controls the spacing inside an element's border.

div { padding: 10px; }

margin

Controls the spacing outside an element's border.

div { margin: 10px; }

width

Sets the width of an element.

div { width: 50%; }

height

Sets the height of an element.

div { height: 100px; }

text-align

Aligns the text content within an element.

p { text-align: center; }

display

Specifies the layout behavior of an element.

div { display: flex; }
